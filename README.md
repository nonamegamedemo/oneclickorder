#Our Demo

###install Guide
```shell
npm install webpack -g
npm install webpack-dev-server -g
npm install
```

###打包
使用 ```npm run build``` 来build项目
开发阶段可以使用 ```npm run watch``` 来监听文件的修改，当文件修改后会自动进行打包操作

###运行
使用 ```npm run start``` 开启服务，通过对应端口可以访问网页
